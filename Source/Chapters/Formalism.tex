\section{Classical Field Theory}

\lipsum

This is an integral $\integrate{d^3x}{\psi( \fv{x}{\mu} )}$.

The wavefunction is

\be
\psi(x) = \integrate{ \frac{d^3p}{(2\pi)^{\frac{3}{2}}}{ \psi(p)} }
        = \integrate{ \frac{d^3p}{(2\pi)^{\frac{3}{2}}}{ \braket{p|\psi} } }
\ee

\be
\psi(\fv{x}{\mu})   = \int dp_{\mu} ~ \psi(\fv{p}{\mu}) e^{-i \fV{p}{\mu}\fv{x}{\mu} }
\ee

The general state $\ket{\psi}$ may be expanded in eigenstates of $\ket{x}$...

\be
\ket{\psi}  = \FT{x}{3} ~ \ket{x}\braket{x|\psi}
            = \FT{x}{3} ~ \psi(\vec{x}) ~ \ket{x}
\ee
